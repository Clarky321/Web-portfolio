1. REACT
React — это библиотека для создания пользовательских интерфейсов.
Она упрощает разработку интерфейсов за счёт компонентов, позволяя разбивать код на небольшие модули.
React хорошо подходит для динамических страниц с возможностью повторного использования компонентов.

2. NEXT.JS
Next.js — это фреймворк для React, который добавляет поддержку рендеринга на стороне сервера (SSR)
и статической генерации (SSG), что значительно улучшает SEO и производительность.
Он также обеспечивает маршрутизацию, позволяя легко создавать многостраничные сайты.

Что такое SSR и SSG?
SSR (Server Side Rendering) генерирует HTML на сервере в реальном времени для каждого запроса пользователя.
Такой процесс обеспечивает возможность обновления контента, но может быть медленным, особенно во время пикового трафика.
SSG (Static Site Generator) создаёт статические HTML-файлы во время компиляции, которые доставляются конечному пользователю.
Благодаря предварительному рендерингу контент обеспечивает очень быструю загрузку.

3. CHAKRA UI
Chakra UI — это библиотека компонентов для React с поддержкой темизации и стилизации.
Она упрощает разработку пользовательского интерфейса, позволяя быстро настраивать стиль и адаптивность.

4. FRAMER MOTION
Framer Motion — это библиотека для анимаций, которая легко интегрируется с React.
Она добавляет плавные анимации для элементов и переходов, что улучшает UX и добавляет интерактивности.

5. THREE.JS
Three.js — это библиотека для 3D-графики в браузере.
Она позволяет добавлять на сайт трёхмерные модели и сцены, что делает сайт визуально более привлекательным и интерактивным.
